# Assets

This folder stores the raw visual resources used by this Sprint 1 repository.

It includes screenshots and supporting files that help document the patient portal concept for planning and design work. These assets are not the implementation itself. The curated explanation of how these images are used lives in [../design-reference/README.md](../design-reference/README.md).

## Included Resources
- `patient-sign-in-screen.png`: main patient sign-in reference.
- `patient-sign-in-screen-v1.png`: earlier sign-in capture kept for comparison.
- `patient-registration-screen.png`: patient account creation reference.
- `password-reset-request-screen.png`: forgot password reference.
- `patient-dashboard-screen.png`: dashboard overview reference.
- `book-appointment-screen.png`: doctor and slot selection reference.
- `my-appointments-screen.png`: appointments management reference.
- `medical-records-screen.png`: medical records reference.
- `profile-screen.png`: patient profile reference.
