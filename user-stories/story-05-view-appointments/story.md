# User Story 5

## User Story
As a patient, I want to view my appointments so that I can track my visits.

## Story Points
3 story points

## Acceptance Criteria
### Patient
- Patients should be able to view a list of their appointments after signing in.
- The appointment list should show key details, such as doctor name, date, time, and status.
- Patients should be able to distinguish between upcoming, completed, and cancelled appointments.
- Patients should be able to open an appointment entry to view more details if needed.
