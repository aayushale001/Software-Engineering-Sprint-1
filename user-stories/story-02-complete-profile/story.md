# User Story 2

## User Story
As a patient, I want to complete my profile so that I can use the full portal.

## Story Points
5 story points

## Acceptance Criteria
### Patient
- Patients should be able to enter and save their personal profile details, such as name, contact details, date of birth, and address.
- Required profile fields should be clearly marked and validated before the profile can be completed.
- Patients should be able to update their profile information later if their details change.
- The portal should indicate whether the patient profile is complete or incomplete.
- Features that depend on profile completion should only become available once the required information has been provided.
