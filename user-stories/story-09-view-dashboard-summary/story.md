# User Story 9

## User Story
As a patient, I want to view a dashboard summary so that I can quickly access key features.

## Story Points
3 story points

## Acceptance Criteria
### Patient
- Patients should see a dashboard after signing in.
- The dashboard should provide a summary of important information, such as upcoming appointments and profile completion status.
- The dashboard should provide quick access links to important portal features, such as booking appointments and viewing medical records.
- If there is no data available for a section, the dashboard should show a clear empty state instead of failing.
