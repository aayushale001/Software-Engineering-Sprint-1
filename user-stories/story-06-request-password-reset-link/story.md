# User Story 6

## User Story
As a patient, I want to request a password reset link so that I can recover my account if I forget my password.

## Story Points
3 story points

## Acceptance Criteria
### Patient
- Patients should be able to request a password reset from the sign-in page.
- Patients should be able to submit their registered email address to request the reset link.
- The system should send a secure password reset link to the registered email address.
- The system should display a generic confirmation message whether or not the email exists in order to protect account privacy.
- Multiple rapid reset requests should be restricted to reduce misuse.
