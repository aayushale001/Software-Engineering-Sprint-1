# User Story 3

## User Story
As a patient, I want to browse doctors and view available slots so that I can choose a suitable appointment.

## Story Points
5 story points

## Acceptance Criteria
### Patient
- Patients should be able to view a list of doctors available on the portal.
- Patients should be able to see key doctor details, such as name, specialty, and availability.
- Patients should be able to view available appointment slots for each doctor by date and time.
- Unavailable or already booked slots should not be shown as selectable.
- Patients should be able to choose a doctor and a suitable slot to continue with booking.
