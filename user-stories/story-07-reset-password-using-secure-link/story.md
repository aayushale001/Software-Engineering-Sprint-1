# User Story 7

## User Story
As a patient, I want to reset my password using a secure link so that I can regain access to my account.

## Story Points
5 story points

## Acceptance Criteria
### Patient
- Patients should be able to open the password reset page only through a valid secure reset link.
- The reset link should expire after a limited time or after it has been used once.
- Patients should be able to enter and confirm a new password that meets the password policy.
- The system should reject expired, invalid, or already used reset links and show a clear message.
- After a successful password reset, the old password should no longer work and the new password should allow sign-in.
